「黑客世界」的入口

对学习渗透测试有帮助的一些学习渠道。

学习渠道

  • Twitter
    • 可以通过关键字搜索到细致的资料/工具
    • 可以和大牛交流
    • 最新漏洞情报
  • YouTube
    • 丰富的视频资源
  • Pentester Land
    • 对渗透测试人员的作用巨大，内含过去到现在所有经典的挖洞过程
  • Bug Bounty Reference
    • 几乎囊括了所有与web安全相关的技术点
  • Payloads All The Things
    • 不管是红队/渗透/ctf玩家,都能在这里找到想要的资料
  • bilibili
    • 对，就是B站，它实际上可以说是国内学习资源最丰富的视频网站，里面有“操作系统原理”、“计算机编程语言”、“计算机科学”、“算法基础”等很多优秀课程，甚至是国外的教程，并且是翻译好的，只需要负责学就行啦~

新闻

  • The Hacker News(国外)
  • FreeBuf(国内)

工具

  • KitPloit
    • 介绍了很多的黑客工具
  • Connect Trojan
    • 一个“RAT”下载器,可以找到A-Z几百款国外的“RAT”,很多APT组织都会大量利用这些
    • 里面很多都是开源工具,可以进行二次开发,自己重新打磨

漏洞利用

  • 关于漏洞的利用, Metasploit 是效率最高的工具。可以在 这里 获得最新的漏洞利用

论坛

  • Offensive Community
  • Cracking

漏洞搜索引擎

  • SPLOITUS

    • 漏洞搜索引擎，可以最快地找到公开的漏洞

知识平台

  • media.ccc.de
    • 由计算机俱乐部分发的各种视频、音频资料
